# AGENTS.md for company dashboard repositories

This repository is one company financial intelligence and business tracking dashboard.

## Standard sections

- 01 核心指标与规模趋势
- 02 增长来源与业务结构
- 03 盈利质量与经营杠杆
- 04 后续观察

## Writing rules

- Use management-briefing Chinese.
- Titles must be judgment statements.
- Each card must connect claim, evidence, and analysis.
- Do not duplicate metric cards if the metric appears in the trend overview chart.
- Use the chart top-right label for source notes.
- Avoid vague translated terms.
- Explain important terms on first use.

## Workflow

- Default branch: main.
- Run npm run build before commit.
- Push origin main only when user asks to deploy.
